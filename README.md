# 🌍 Climate Change Prediction Using Machine Learning

## 📌 Project Overview

This project analyzes climate change trends and predicts temperature variations using Machine Learning models.
The goal is to identify environmental factors influencing global temperature change.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Conduct Exploratory Data Analysis (EDA)
* Visualize climate trends
* Train Machine Learning models
* Predict temperature variations

## Research Objective
This project explores patterns and predictive modeling techniques to understand real-world phenomena using data-driven approaches.

## ✅ Results

The Random Forest model achieved strong prediction performance and identified CO₂ emissions as the most influential feature affecting temperature trends.

Models Used:
- Linear Regression
- Random Forest Regressor

### Model Performance

| Model | R² Score |
|------|---------|
| Linear Regression | 0.006 |
| Random Forest | -0.07 |

Mean Squared Error: 74.16

### Key Insights
- Climate temperature prediction shows high variability.
- Linear Regression performed slightly better than Random Forest.
- Results highlight the complexity of long-term climate prediction problems.
