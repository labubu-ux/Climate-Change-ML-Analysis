# 🌍 Climate Change Prediction Using Machine Learning

## 📌 Project Overview

This project analyzes climate change trends and predicts temperature variations using Machine Learning models.
The goal is to identify environmental factors influencing global temperature change.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Conduct Exploratory Data Analysis (EDA)
* Visualize climate trends
* Train Machine Learning models
* Predict temperature variations

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

---

## 📊 Dataset Features

* CO2 Emissions
* Methane Levels
* Nitrous Oxide
* Energy Consumption
* Temperature Change

---

## 🤖 Machine Learning Models

* Linear Regression
* Random Forest Regressor

---

## 📈 Key Visualizations

* Correlation Heatmap
* Feature Importance
* Prediction Graphs

---

## ✅ Results

The Random Forest model achieved strong prediction performance and identified CO₂ emissions as the most influential feature affecting temperature trends.
